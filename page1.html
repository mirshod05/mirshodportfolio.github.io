Class Projects

I have done a few class projects throughout my time at Wayne State. Here are some of them.

My group and I built a centrifuge that can be used to spin samples with its base made from Siemens NX 3D printing, as well as Arduino code to get the motor spinning. You can see the images below.

  • As part of an Honors Project for my Data Structures class, I got to work with grid computing. I created a matrix multiplication program and ran it in the university's grid and execute it in parallel under different numbers of threads to observe the patterns between execution time and the number of threads. You can view the report here.
